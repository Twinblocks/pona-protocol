# PoNa – Proof of Node Activity

PoNa (Proof of Node Activity) is a participation-based consensus and reward model designed to incentivize meaningful node contribution in decentralized networks.

## Concept

Unlike Proof of Work or Proof of Stake, PoNa rewards nodes based on measurable activity such as:

- node uptime
- transaction relay
- governance participation
- network services
